Recent research projects and associated code.

RENN repository


An inference neural network for generic inference and learning of Markov random fields (MRFs). The model minimizes region-based free energy directly by amortizing the beliefs of regions. The method is an alternative to the classical message-passing algorithm, e.g., belief propagation and generalized belief propagation. This implementation uses the PyTorch framework in python and supports inference with GPU.

GenHMM repository


A neural network based hidden Markov model. GenHMM is a generator-mixed (modeled by neural networks) hidden Markov model, which employs the expressive power of networks and meanwhile keeps the tractability of likelihood via introducing normalizing flows in the model. GenHMM supports comment probabilistic inference tasks of sequential or temporal signals. GenHMM can be learned from data within expectation maximization. Check out the application of GenHMM to speech recognition and sepsis detection of infants.

GenMM repository


GenMM is a mixture model of normalizing flows that can be learned within expectation maximization algorithm. GenMM allows the exact likelihood computation of evidence and efficient sample generation. The python implementation of the model is available in the repository.

AlphaBP repository


AlphaBP is a variant of the message-passing algorithm. It supports generic inference tasks for problems described by graphical models, which is an add to your toolbox of message-passing algorithms, e.g., belief propagation or sum-product. The repository above contains an implementation of the algorithm in python.

EOTGM repository

EOTGM is an implicit generative model, where probabilistic density function is not explicitly available. Entropy-regularized optimal transport (EOT) cost is employed in developing generative models to learn implicit distributions. In model learning, it is optional to either use EOT cost directly or use EOT cost iteratively in an adversarial game.

Pyro

A Python library for probabilistic modeling, inference, and criticism.

Edward

A universal probabilistic programming language (PPL) written in Python and supported by PyTorch on the backend.

pgmpy

A python library for working with Probabilistic Graphical Models.

Anglican

A probabilistic programming language integrated with Clojure and ClojureScript.
